## 何も見ずに解いてみる

問題文の意味がよくわからなかった
下はとりあえず左と右のsubtreeの深さが同じな木を作ればいいと思って作ったコード

```python
class Solution:
    def sortedArrayToBST(self, nums: List[int]) -> Optional[TreeNode]:
        if not nums:
            return None
        num_stack = nums[:]
        root = TreeNode(num_stack.pop())
        node_queue = [root]
        while node_queue:
            node = node_queue.pop()
            if num_stack:
                node.left = TreeNode(num_stack.pop())
                node_queue.append(node.left)
            if num_stack:
                node.right = TreeNode(num_stack.pop())
                node_queue.append(node.right)
        return root
```

## 色々調べてみる

どうやらBSTというものは 左の子 < 親 < 右の子 となっているような木のことを言うらしい。
二分探索ができ、挿入・削除もlogNのオーダーでできるデータ構造。

stackを使うやり方

```python
class Solution:
    def sortedArrayToBST(self, nums: List[int]) -> Optional[TreeNode]:
        if not nums:
            return None
        root = TreeNode()
        stack = [(root, 0, len(nums))]
        while stack:
            node, left, right = stack.pop()
            mid = (left + right) // 2
            node.val = nums[mid]
            if mid != left:
                node.left = TreeNode()
                stack.append((node.left, left, mid))
            if mid + 1 != right:
                node.right = TreeNode()
                stack.append((node.right, mid + 1, right))
        return root
```

## 最終コード

```python
class Solution:
    def sortedArrayToBST(self, nums: List[int]) -> Optional[TreeNode]:
        def construct_BST_in_range(left, right):
            if left >= right:
                return None
            mid = (left + right) // 2
            root = TreeNode(nums[mid])
            root.left = construct_BST_in_range(left, mid)
            root.right = construct_BST_in_range(mid + 1, right)
            return root
        return construct_BST_in_range(0, len(nums))
```
