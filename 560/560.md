## 何も見ずに解いてみる

```python
from collections import defaultdict
from itertools import chain

class Solution:
    def subarraySum(self, nums: List[int], k: int) -> int:
        subarray_count = 0
        cumsum_to_count = defaultdict(int)
        cumsum = 0
        for num in chain([0], nums):
            cumsum += num
            target = cumsum - k
            if target in cumsum_to_count:
                subarray_count += cumsum_to_count[target]
            cumsum_to_count[cumsum] += 1
        return subarray_count
```

最初辞書への追加を全部走らせてからfor cumsum in cumsum_to_countでまとめてチェックする形にしていたが、走っているnumのindexより後の累積和もチェックすることになってしまいうまくいかなかった。
配列が全部正の数のものだったらこれでもうまくいくかも。

実装後に気付いたがこのif文はfor cumsum in cumsum_to_countを使わないなら要らないか。
```python

```
このcumsum_to_countをfor分で回す形だと、このif文を失くしてしまうと"dictionary size changed during iteration"というエラーが出てしまった。（cumsum_to_count[target]を評価した段階でその結果が0であっても辞書にそのtargetが追加されてしまうため）

`cumsum_to_count[current_cumsum] += 1`はfor文の最後に持ってこないとk=0の場合に今追加したばかりのcumsumをチェックしてしまう。

from collections import Counter
from itertools import accumulate

class Solution:
    def subarraySum(self, nums: List[int], k: int) -> int:
        subarray_count = 0
        cumsums = list(accumulate(nums, initial=0))
        cumsum_to_count = Counter(cumsums)
        for cumsum in cumsums:
            cumsum_to_count[cumsum] -= 1
            target = k + cumsum
            subarray_count += cumsum_to_count[target]
        return subarray_count

## 最終コード

cumsum_to_countに0を追加するための行をどう書くかは迷った。`cumsum_to_count[cumsum] += 1`にしたが、+= 1を= 1にする選択肢もあるしcumsumを0にする選択肢もありそう。
個人的には`cumsum_to_count[0]`とすると0がマジックナンバーっぽく映ってしまう気がしたのでcumsumとした。

```python
from collections import defaultdict

class Solution:
    def subarraySum(self, nums: List[int], k: int) -> int:
        subarray_count = 0
        cumsum = 0
        cumsum_to_count = defaultdict(int)
        cumsum_to_count[cumsum] += 1
        for num in nums:
            cumsum += num
            target = cumsum - k
            subarray_count += cumsum_to_count[target]
            cumsum_to_count[cumsum] += 1
        return subarray_count
```
