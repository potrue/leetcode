## 何も見ずに解いてみる

```python
class Solution:
    def numUniqueEmails(self, emails: List[str]) -> int:
        unique_emails = set()
        for email in emails:
            local, domain = email.split('@')
            local = local.split('+', 1)[0]
            local = local.replace(".", "")
            unique_emails.add(local + '@' + domain)
        return len(unique_emails)
```

splitやreplaceは使い方を覚えてなかったのでドキュメントを読んで書いたのですが、このぐらいの関数だったら覚えている人も多いんですかね。

一文字ずつ走査していくやり方でも作ってみた。（match caseとかstateのLiteralとか色々慣れないものを使ってみたんですがどうでしょうか）
str += char の書き方は新しく文字列を作るため今まで出来ている文字列の長さ分の計算が余計に発生すると思われるので一度listに格納していく方法にしてみた。
（でも試してみたらどちらもそんなに時間変わらなかったです）

```python
class Solution:
    def numUniqueEmails(self, emails: List[str]) -> int:
        def normalize(email):
            state: Literal["on_local", "on_tag", "on_domain"] = "on_local"
            valid_characters = []
            for char in email:
                match char:
                    case "@":
                        state = "on_domain"
                        valid_characters.append(char)
                    case "+":
                        if state == "on_local":
                            state = "on_tag"
                    case ".":
                        if state == "on_domain":
                            valid_characters.append(char)
                    case _:
                        if state == "on_local" or state == "on_domain":
                            valid_characters.append(char)
            normalized_email = "".join(valid_characters)
            return normalized_email

        unique_emails = set()
        for email in emails:
            unique_emails.add(normalize(email))
        return len(unique_emails)
```

## 最終コード

@がなかった場合にnot enough values to unpackのエラーを吐くのがちょっといやかなと思ったのでpartitionメソッドというのを使ってみた。
https://docs.python.org/3/library/stdtypes.html#str.partition

```python
class Solution:
    def numUniqueEmails(self, emails: List[str]) -> int:
        unique_emails = set()
        for email in emails:
            local, at, domain = email.partition("@")
            local = local.split("+")[0].replace(".", "")
            unique_emails.add(local + at + domain)
        return len(unique_emails)
```
