## 何も見ずに解いてみる

そもそもinorderとpreorderが何かよくわからないので調べる
pre-order: すべてのノードについてそのノード → 左側のsubtree → 右側のsubtreeという探索の順番になるような順番（leftを先に探索するDFSの順番）
in-order: すべてのノードについて左側のsubtree → そのノード → 右側のsubtreeという探索の順番になるような順番
post-order: すべてのノードについて左側のsubtree → 右側のsubtree → そのノードという探索の順番になるような順番

結構難しかった。手作業でやるならどういうやり方ができるかを想像してそのまま変換したら下のコードが出来上がった。
iを探すところは二分探索にしてもよさそう。division_nodesにinsertする部分が計算量的にはネックになっていそう

```python
class Solution:
    def buildTree(self, preorder: List[int], inorder: List[int]) -> Optional[TreeNode]:
        val_to_inorder_index = dict()
        for idx, val in enumerate(inorder):
            val_to_inorder_index[val] = idx
        sentinel = TreeNode(None)
        division_nodes = [sentinel]
        val_to_inorder_index[None] = float("inf")
        for val in preorder:
            node = TreeNode(val)
            i = 0
            while i < len(division_nodes) and val_to_inorder_index[val] > val_to_inorder_index[division_nodes[i].val]:
                i += 1
            if i == 0:
                division_nodes[0].left = node
            else:
                division_nodes[i - 1].right = node
            division_nodes.insert(i, node)
            division_nodes = division_nodes[i:]
        return sentinel.left
```

## 最終コード

計算量をO(N)に抑える方法だとなかなかしっくりくる書き方が見つからない、、

```python
class Solution:
    def buildTree(self, preorder: List[int], inorder: List[int]) -> Optional[TreeNode]:
        val_to_inorder_idx = {val: idx for idx, val in enumerate(inorder)}
        def build_subtree(preorder_begin, inorder_begin, tree_size):
            if tree_size == 0:
                return None
            root_val = preorder[preorder_begin]
            left_size = val_to_inorder_idx[root_val] - inorder_begin
            right_size = tree_size - left_size - 1
            root = TreeNode(root_val)
            root.left = build_subtree(
                preorder_begin + 1,
                inorder_begin,
                left_size
            )
            root.right = build_subtree(
                preorder_begin + 1 + left_size,
                inorder_begin + left_size + 1,
                right_size
            )
            return root
        return build_subtree(0, 0, len(preorder))
```
