queueを二つ用意してやろうとしてこんがらがったので再帰にした。

```python
class Solution:
    def mergeTrees(self, root1: Optional[TreeNode], root2: Optional[TreeNode]) -> Optional[TreeNode]:
        if root1 and root2:
            root1.val += root2.val
            root1.left = self.mergeTrees(root1.left, root2.left)
            root1.right = self.mergeTrees(root1.right, root2.right)
            return root1
        if root1 and not root2:
            return root1
        if not root1 and root2:
            return root2
        if not root1 and not root2:
            return None
```

stackで書くとしたらこんな感じだけど場合分けとかフィルターが多くなってしまう。

```python
class Solution:
    def mergeTrees(self, root1: Optional[TreeNode], root2: Optional[TreeNode]) -> Optional[TreeNode]:
        if not root1 and not root2:
            return None
        stack = []
        merged_root = TreeNode()
        stack.append((merged_root, list(filter(None, [root1, root2]))))
        while stack:
            merged_node, nodes = stack.pop()
            merged_node.val = sum([node.val for node in nodes])
            lefts = list(filter(None, [node.left for node in nodes]))
            rights = list(filter(None, [node.right for node in nodes]))
            if lefts:
                merged_node.left = TreeNode()
                stack.append((merged_node.left, lefts))
            if rights:
                merged_node.right = TreeNode()
                stack.append((merged_node.right, rights))
        return merged_root
```

## 最終コード

```python
class Solution:
    def mergeTrees(self, root1: Optional[TreeNode], root2: Optional[TreeNode]) -> Optional[TreeNode]:
        if not root1 and not root2:
            return None
        if not root1:
            root1 = TreeNode()
        if not root2:
            root2 = TreeNode()
        merged_root = TreeNode()
        merged_root.val = root1.val + root2.val
        merged_root.left = self.mergeTrees(root1.left, root2.left)
        merged_root.right = self.mergeTrees(root1.right, root2.right)
        return merged_root
```
