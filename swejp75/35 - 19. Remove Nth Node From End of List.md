### Solution 1

```python
class Solution:
    def removeNthFromEnd(self, head: Optional[ListNode], n: int) -> Optional[ListNode]:
        size = 0
        node = head
        while node:
            size += 1
            node = node.next
        steps = size - n
        if steps == 0:
            return head.next
        node = head
        for _ in range(steps - 1):
            node = node.next
        node.next = node.next.next
        return head
```

### Solution 2

stackを使わずに Follow up: Could you do this in one pass? を満たす方法

```python
class Solution:
    def removeNthFromEnd(self, head: Optional[ListNode], n: int) -> Optional[ListNode]:
        if n <= 0:
            raise ValueError("invalid n")
        scout = head
        for _ in range(n):
            if not scout:
                raise ValueError("invalid n")
            scout = scout.next
        sentinel = ListNode(next=head)
        node = sentinel
        while scout is not None:
            node = node.next
            scout = scout.next
        node.next = node.next.next
        return sentinel.next
```
