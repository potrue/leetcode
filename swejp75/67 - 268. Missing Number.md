### Solution 1

```python
class Solution:
    def missingNumber(self, nums: List[int]) -> int:
        return sum(range(len(nums) + 1)) - sum(nums)
```

### Solution 2

```python
class Solution:
    def missingNumber(self, nums: List[int]) -> int:
        cumulative_xor = 0
        for i in range(len(nums) + 1):
            cumulative_xor ^= i
        for num in nums:
            cumulative_xor ^= num
        return cumulative_xor
```
