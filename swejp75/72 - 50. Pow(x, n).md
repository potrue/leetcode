### Solution 1

```python
class Solution:
    def myPow(self, x: float, n: int) -> float:
        if n < 0:
            return 1 / self.myPow(x, -n)
        result = 1
        while n:
            if n & 1:
                result *= x
            x *= x
            n //= 2
        return result
```
