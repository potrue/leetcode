### Solution 1

時間計算量nlogn

```python
class Solution:
    def topKFrequent(self, nums: List[int], k: int) -> List[int]:
        num_to_count = Counter(nums)
        sorted_by_count = sorted(num_to_count.keys(), key=num_to_count.get, reverse=True)
        return sorted_by_count[:k]
```

### Solution 2

時間計算量nlogk

```python
class Solution:
    def topKFrequent(self, nums: List[int], k: int) -> List[int]:
        num_to_count = Counter(nums)
        top_k_heap = []
        for num, count in num_to_count.items():
            if len(top_k_heap) < k:
                heapq.heappush(top_k_heap, (count, num))
            else:
                heapq.heappushpop(top_k_heap, (count, num))
        return list(reversed([num for count, num in top_k_heap]))
```
