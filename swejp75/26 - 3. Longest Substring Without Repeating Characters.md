### Solution 1

```python
class Solution:
    def lengthOfLongestSubstring(self, s: str) -> int:
        chars = set()
        start = 0
        result = 0
        for c in s:
            while c in chars:
                chars.remove(s[start])
                start += 1
            chars.add(c)
            result = max(result, len(chars))
        return result
```
