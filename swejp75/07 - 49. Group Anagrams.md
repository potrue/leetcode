### Solution 1

```python
class Solution:
    def groupAnagrams(self, strs: List[str]) -> List[List[str]]:
        group_to_strings = defaultdict(list)
        for s in strs:
            group = "".join(sorted(s))
            group_to_strings[group].append(s)
        return list(group_to_strings.values())
```
