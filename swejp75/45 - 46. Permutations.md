### Solution 1

```python
class Solution:
    def permute(self, nums: List[int]) -> List[List[int]]:
        nums = nums[:]
        result = []
        def permuteHelper(index):
            if index == len(nums):
                result.append(nums[:])
            for i in range(index, len(nums)):
                nums[i], nums[index] = nums[index], nums[i]
                permuteHelper(index + 1)
                nums[i], nums[index] = nums[index], nums[i]
        permuteHelper(0)
        return result
```
