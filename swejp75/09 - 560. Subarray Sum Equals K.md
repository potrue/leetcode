### Solution 1

```python
class Solution:
    def subarraySum(self, nums: List[int], k: int) -> int:
        result = 0
        cumsum_to_count = defaultdict(int)
        cumsum_to_count[0] += 1
        cumsum = 0
        for num in nums:
            cumsum += num
            result += cumsum_to_count[cumsum - k]
            cumsum_to_count[cumsum] += 1
        return result
```
