### Solution 1

```python
class Solution:
    def minEatingSpeed(self, piles: List[int], h: int) -> int:
        low = 1
        high = max(piles)
        while low < high:
            speed = (low + high) // 2
            hours_required = 0
            for pile in piles:
                hours_required += math.ceil(pile / speed)
            if hours_required <= h:
                high = speed
            else:
                low = speed + 1
        return low
```

math.ceilの代替として、(a + b - 1) // b という式があるみたい。これは覚えておいても良さそう。
