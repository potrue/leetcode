## ステップ1: 何も見ずに解く

```python
class Solution:
    def detectCycle(self, head: Optional[ListNode]) -> Optional[ListNode]:
        visited = set()
        current = head
        while current:
            if current in visited:
                return current
            visited.add(current)
            current = current.next
        return None
```

素直に実装するとしたらこうだろうか（時間計算量O(N), 空間計算量O(N)）

- 二つ目のwhileを`while slow is not head`で回すか、`while True`の中に`if slow is fast: return slow`を入れて回すか
  - `while True`はまかり間違って無限ループに入ることがありそうでなんか怖い気がするが、`while slow is not head`でもよく考えたら別に怖い気がする（まあもっとよくよく考えるとどっちも怖くないんでしょうけど）

あたりを考えた結果次のコードにした

```python
class Solution:
    def detectCycle(self, head: Optional[ListNode]) -> Optional[ListNode]:
        slow = head
        fast = head

        while fast and fast.next:
            slow = slow.next
            fast = fast.next.next
            if fast is slow:
                break
        else:
            return None

        while slow is not head:
            slow = slow.next
            head = head.next
            
        return slow
```
