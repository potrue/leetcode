配列がソートされていればいろいろと解法にも幅が出そう。
せっかくなのでソートされた状態を想定したコードを書いてみた。

```python
class Solution:
    def intersection(self, nums1: List[int], nums2: List[int]) -> List[int]:
        nums1, nums2 = sorted(nums1), sorted(nums2)
        intersection_list = []
        i1 = i2 = 0
        while i1 < len(nums1) and i2 < len(nums2):
            n1, n2 = nums1[i1], nums2[i2]
            if n1 < n2:
                i1 += 1
                continue
            if n1 > n2:
                i2 += 1
                continue
            if n1 == n2:
                intersection_list.append(n1)
                while i1 < len(nums1) and n1 == nums1[i1]:
                    i1 += 1
                while i2 < len(nums2) and n2 == nums2[i2]:
                    i2 += 1
        return intersection_list
```

## 最終コード

```python
class Solution:
    def intersection(self, nums1: List[int], nums2: List[int]) -> List[int]:
        set_ = set(nums1) & set(nums2)
        return list(set_)
```
