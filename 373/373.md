## 何も見ずに解く

全てのペアを順番に要素数がkのheapに入れていくやり方（時間計算量O(n^2 log k?)しか思い付かずTime Limit Exceededになってしまったので解答を見た。

https://leetcode.com/problems/find-k-pairs-with-smallest-sums/solutions/84550/slow-1-liner-to-fast-solutions
解答はこちらが分かりやすかった。次なる最小値となり得る候補をheapに入れていって、一つずつ取り出していくという方法。
このページにあるような2次元のイメージを作るとよく理解できた。

```python
class Solution(object):
    def kSmallestPairs(self, nums1, nums2, k):
        queue = []
        def try_push(i, j):
            if i < len(nums1) and j < len(nums2):
                heapq.heappush(queue, (nums1[i] + nums2[j], i, j))
        try_push(0, 0)
        pairs = []
        added_pairs_set = set()
        while len(pairs) < k:
            _, i, j = heapq.heappop(queue)
            pairs.append([nums1[i], nums2[j]])
            added_pairs_set.add((i, j))
            if i == 0 or (i-1, j+1) in added_pairs_set:
                try_push(i, j+1)
            if j == 0 or (i+1, j-1) in added_pairs_set:
                try_push(i+1, j)
        return pairs
```

## 最終コード

```python
class Solution(object):
    def kSmallestPairs(self, nums1, nums2, k):
        candidates = []
        def try_push(i, j):
            if i < len(nums1) and j < len(nums2):
                heapq.heappush(candidates, (nums1[i] + nums2[j], i, j))
        try_push(0, 0)
        pairs = set()
        while candidates and len(pairs) < k:
            _, i, j = heapq.heappop(candidates)
            pairs.add((i, j))
            if i == 0 or (i-1, j+1) in pairs:
                try_push(i, j+1)
            if j == 0 or (i+1, j-1) in pairs:
                try_push(i+1, j)
        return [[nums1[i], nums2[j]] for i, j in pairs]
```

（かかった時間: 3:08, 3:10, 2:37）

ちなみに、
```python
if i == 0 or (i-1, j+1) in pairs:
    try_push(i, j+1)
if j == 0 or (i+1, j-1) in pairs:
    try_push(i+1, j)
```
この部分は単純に
```python
try_push(i, j+1)
if j == 0:
    try_push(i+1, j)
```
としてしまった方が実行速度が早かった。最適化の恩恵よりifを回す回数を減らしたほうが良いということだろう。
考えてみれば時間計算量O(k log k)のlogの中身の部分にしかこの最適化は効かないので、効果が少ないのは納得できる。
この方がsetを別に用意する必要もないので良い気もするが、アルゴリズム的にはnums1とnums2が非対称になるので少しもどかしい感じが・・・
