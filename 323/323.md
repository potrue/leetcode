class Solution:
    def countComponents(self, n: int, edges: List[List[int]]) -> int:
        node_to_connected = {node: set() for node in range(n)}
        for node1, node2 in edges:
            node_to_connected[node1].add(node2)
            node_to_connected[node2].add(node1)
        num_components = 0
        seen_nodes = set()
        for node in range(n):
            if node not in seen_nodes:
                num_components += 1
            stack = [node]
            seen_nodes.add(node)
            while stack:
                stacked = stack.pop()
                for connected in node_to_connected[stacked]:
                    if connected not in seen_nodes:
                        stack.append(connected)
                        seen_nodes.add(connected)
        return num_components

## 最終コード

変数名が結構難しい。
nodeとすると数字以外の情報も持っているべきな気がするが（でもこれは最初の方に解いたlinked list関連の問題のせいかもしれない）、node_idやnode_numberとすると冗長な気がしないでもない・・・  
Adjacency listは言葉として割と定着しているものらしい。  
https://en.wikipedia.org/wiki/Adjacency_list  

```python
class Solution:
    def countComponents(self, n: int, edges: List[List[int]]) -> int:
        adjacency_list = {[] for _ in range(n)}
        for v1, v2 in edges:
            adjacency_list[v1].append(v2)
            adjacency_list[v2].append(v1)

        visited_vertices = set()
        def traverse_component(starting_vertex):
            stack = [starting_vertex]
            visited_vertices.add(starting_vertex)
            while stack:
                vertex = stack.pop()
                for adjacent in adjacency_list[vertex]:
                    if adjacent not in visited_vertices:
                        stack.append(adjacent)
                        visited_vertices.add(adjacent)

        num_components = 0
        for vertex in range(n):
            if vertex in visited_vertices:
                continue
            num_components += 1
            traverse_component(vertex)
        return num_components
```
