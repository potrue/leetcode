## ステップ1: 何も見ずに解く

```python
class Solution:
    def deleteDuplicates(self, head: Optional[ListNode]) -> Optional[ListNode]:
        first_node_of_value = ListNode(None)
        node = head
        while node:
            if node.val != first_node_of_value.val:
                first_node_of_value.next = node
                first_node_of_value = node
            node = node.next
        first_node_of_value.next = None
        return head
```

## ステップ2: いろいろ調べてみる

他の人の実装をいくつか読んでみて、次の実装が一番わかりやすく感じた。

```python
class Solution:
    def deleteDuplicates(self, head: Optional[ListNode]) -> Optional[ListNode]:
        node = head
        while node and node.next:
            if node.val == node.next.val:
                node.next = node.next.next
            else:
                node = node.next
        return head
```

次の実装も悪くないような気がしたが、ポインタが二つ同時に登場する分、頭の中で想像しようとするとちょっと難しくなるので、上の方が好みかもしれない。

```python
class Solution:
    def deleteDuplicates(self, head: Optional[ListNode]) -> Optional[ListNode]:
        node = head
        while node:
            next_node = node.next
            while next_node and node.val == next_node.val:
                next_node = next_node.next
            node.next = next_node
            node = next_node
        return head
```

たぶん、本質的にはLinkedListの構造は前（next）方向を向いている気持ちの実装なのにも関わらず、ステップ1の実装では「前の方のノードの立場になって、後ろの方のノードと値が違ったら処理をする」という書き方になってしまっているせいで、
前を向いたり（nodeを前に進める時）後を向いたり（first_node_of_valueをチェックする時）することになってしまっていたり、first_node_of_valueの初期化の変数に気を使わないといけなくなっていたり、
first_node_of_value.next = Noneを最後に入れる必要が出てきてしまったりしており、それがコードをわかりにくくしてしまっている気がする。

あと、Noneが入力として入ってきた時のために、
first_node_of_value = ListNode(None)
のような回りくどいことをするぐらいであれば、

```python
if not head:
    return None
    
first_node_of_value = None
```

のような感じで早めに返してしまってNoneで初期化したほうが読みやすいかもな、と思った。

## ステップ3: 最終コード（エラーを出さずに3回書いてみる）

```python
class Solution:
    def deleteDuplicates(self, head: Optional[ListNode]) -> Optional[ListNode]:
        node = head
        while node and node.next:
            if node.val == node.next.val:
                node.next = node.next.next
            else:
                node = node.next
        return head
```
