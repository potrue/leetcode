```python
class Solution:
    def isValidBST(self, root: Optional[TreeNode]) -> bool:
        if not root:
            return False
        queue = [(root, float("-inf"), float("inf"))]
        while queue:
            node, expected_min, expected_max = queue.pop()
            if not expected_min < node.val < expected_max:
                return False
            if node.left:
                queue.append((node.left, expected_min, node.val))
            if node.right:
                queue.append((node.right, node.val, expected_max))
        return True
```

左から順番にチェックしていくやり方は、左をチェックしたかどうかをsetで管理するやり方でもできるかも（でも結構わかりづらい）

```python
class Solution:
    def isValidBST(self, root: Optional[TreeNode]) -> bool:
        if not root:
            return False
        checked_left = set()
        prev_val = float("-inf")
        stack = [root]
        while stack:
            node = stack[-1]
            if node not in checked_left:
                if node.left:
                    stack.append(node.left)
                checked_left.add(node)
                continue
            if node.val <= prev_val:
                return False
            prev_val = node.val
            stack.pop()
            if node.right:
                stack.append(node.right)
        return True
```

## 最終コード

```python
class Solution:
    def isValidBST(self, root: Optional[TreeNode]) -> bool:
        stack = [(root, float("-inf"), float("inf"))]
        while stack:
            node, expected_min, expected_max = stack.pop()
            if not node:
                continue
            if not expected_min < node.val < expected_max:
                return False
            stack.append((node.left, expected_min, node.val))
            stack.append((node.right, node.val, expected_max))
        return True
```
